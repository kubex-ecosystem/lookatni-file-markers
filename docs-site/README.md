# LookAtni Docs (MkDocs + Material)

Developer quickstart to work on the documentation site.

## Requirements

- Python 3.8+
- `uv` (modern Python package manager)

## Setup

```bash
cd docs-site
uv sync
uv run mkdocs serve
# open http://localhost:8000
```

## Build

```bash
uv run mkdocs build
```
